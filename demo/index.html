ng-json-viewer

module for Angular.js

JSON Viewer features:

  • Code folding
  • Syntax coloring
  • Display array indicies
  • Display value type

Demo:

Toggle Editor Test Data

Installation:

Via bower (preferred way):
        bower install --save jv.json-viewer
Download zip package:
Download

Basic usage:

  1. Add module as dependency to your app: 
                angular.module('your-app', ['jv.json-viewer']);
  2. Add jsonViewer.css to head of your project: 
                <link rel="stylesheet" href="../dist/jsonViewer.css" />
  3. Add JSON Viewer directive to template: 
                <jv-json-viewer data="yourData" editor="false"></jv-json-viewer>
  4. Full API of JSON Viewer at your disposal.
