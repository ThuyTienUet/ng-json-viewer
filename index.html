ng-json-viewer

An AngularJS directive for viewing JSON.

View on GitHub Download .zip Download .tar.gz

Changes history you can find here

Main features

  • Code folding
  • Array index
  • Syntax highlighting
  • Display value type

Demo

You can see demo on this page

Installation

  • Via bower (preferred way)
bower install --save jv.json-viewer
  • Download zip package

Usage

  1. Add module as dependency to your app:
angular.module('your-app', ['jv.json-viewer']);
  1. Add json viewer directive to template:
<jv-json-viewer></jv-json-viewer>
  1. Default JSON Viewer code colors you can find in demo/jsonViewer.less (at the top of file).

Module's components

  • jsonViewer directive - directive for adding progress bar to the page.
  • jsonViewer service - all directives parsing is done in this service.

jsonViewer directive

Directive will be replaced by its template. For customization different progress bars you can add classes to it. It will be merged with template classes.
